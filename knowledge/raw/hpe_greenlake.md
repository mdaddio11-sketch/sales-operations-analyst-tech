# HPE GreenLake - Cloud Services Platform

Source: https://www.hpe.com/us/en/greenlake.html
Date: 2026-05-04

HPE GreenLake is HPE flagship cloud services platform that brings the cloud experience to apps and data everywhere - in data centers, at the edge, and in co-location facilities. GreenLake offers a pay-per-use model that allows organizations to consume IT infrastructure as a service.

Key features:
- Unified hybrid cloud platform
- Pay-per-use consumption model
- Managed services included
- Available for compute, storage, networking, and HPC workloads
- Supports on-premises, co-location, and edge deployments
